# Zed IDL Extension

This extension provides language support for IDL in the [Zed](https://zed.dev/)
editor.

## Features

- Syntax Highlighting (via `tree-sitter-idl`)
- Code Navigation, Autocompletion, Diagnostics (via `idl-language-server`)

## Installation

Currently, you can install this extension locally by running
`zed: install dev extension` from the Command Palette, and selecting the
`zed-extension` directory.
